# exam_schedule

**Exam Schedule** is a Flutter mobile application designed to help students at FINKI track and manage their exam schedules with ease. The app displays a comprehensive list of upcoming and past exams with detailed information including exam dates, times, room locations, and a countdown showing how much time remains until each exam. Built with Material Design 3 and Flutter, the application provides an intuitive interface with visual distinction between past and future exams to keep students organized and prepared.
